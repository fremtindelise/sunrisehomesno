# Sunrise Homes hjemmeside

Bygget med Next.js.

## Komandoer

Installer avhengigheter: `pnpm install`

Lokal utvikling: `pnpm dev`

Bygg prosjektet: `pnpm build`

Start produksjonsbygg lokalt: `pnpm start`

## Miljøvariabler

Kopier `.env.example` til `.env.local` og fyll inn nødvendige verdier.

## Epost

Prosjektet bruker Scaleway TEM for epost sending fra kontakt skjemaet. `CONTACT_TO_EMAIL` settes som mottaker.

## Oversettelse

Prosjektet bruker Amazon Translate.
